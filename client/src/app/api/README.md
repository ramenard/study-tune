# Client API généré

Code généré — ne pas éditer.

Régénérer avec `npm run generate:api` (API démarrée sur http://localhost:3001).

La spec OpenAPI provient de `http://localhost:3001/docs-json`. Les chemins générés
incluent déjà le préfixe global `/api`, c'est pourquoi le `rootUrl` du client est
configuré à `''` dans `app.config.ts` via `provideApiConfiguration('')`.
